DatabaseTable Class

A data class representing a database table. Specifies the table name and whitelisted columns.

Create a new DatabaseTable instance.

A DatabaseTable object is used to encapsulate information about a particular table in your database. UserFrosting expects all tables to have an id column as their primary key. You can register it with the main data model by using Database::setSchemaTable().

Parameters

$name
the name of the table in your database.
$columns
a list of the column names, not including the id column (which is already assumed to exist).
